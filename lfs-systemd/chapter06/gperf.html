6.37. Gperf-3.1

Gperf 为键集合生成完美的哈希函数。

大致构建用时: 少于 0.1 SBU
所需磁盘空间: 6.3 MB

6.37.1. 安装 Gperf

准备编译 Gperf： 

./configure --prefix=/usr --docdir=/usr/share/doc/gperf-3.1

编译软件包： 

make

该测试已知在运行多任务同时测试（即 -j 选项大于 1）时会失败。用以下命令测试结果： 

make -j1 check

安装软件包： 

make install

6.37.2. Gperf 软件包内容

安装的程序: gperf
安装目录: /usr/share/doc/gperf-3.1

简要介绍
