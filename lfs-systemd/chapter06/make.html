6.66. Make-4.2.1

Make 软件包包含一个用于编译软件的程序。

大致构建用时: 0.6 SBU
所需磁盘空间: 13 MB

6.66.1. 安装 Make

再来一次，解决由 glibc-2.27 或更新版本 glibc 导致的错误： 

sed -i '211,217 d; 219,229 d; 232 d' glob/glob.c

准备编译 Make： 

./configure --prefix=/usr

编译软件包： 

make

测试套件需要知道支持 perl 文件的位置。我们使用一个环境变量来达成这个目的。用以下命令测试结果： 

make PERL5LIB=$PWD/tests/ check

安装软件包： 

make install
